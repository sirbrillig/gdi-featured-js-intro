The DOM Tree: Sample Code

Oftentimes, people think of HTML as something flat -- a bunch of text with tags in the middle. 

<!DOCTYPE html>
<html>
  <head>
    <title>Test Page</title>
    <style>
      h1 {
        color: red;
      }
    </style>
  </head>
  <body>
    <h1>My Page</h1>
    <p>Hello World!</p>
    <img src="http://placekitten.com/g/200/300" alt="cat"/>
  </body>
</html>

The DOM Tree: Sample Model

However, it is something much more than that. Any HTML document is a tree structure defined by the DOM (Document Object Model).

DOM Access

Your browser automatically loads the content of a webpage into a Document object which serves as the entry point into a web page's content.


Using the document you can:

  1. Change the content tree any way you want.
  2. Build an HTML document from scratch.
  3. Access or replace any existing DOM nodes (HTML elements in the DOM).
### The Web API ```js function showCat() { const cat = window.document.querySelector('.kitten-pic'); cat.style.visibility = 'visible'; } ```
### The Web API When using JavaScript inside a browser, you can access web things that are not part of the language. These are called the "Web API".
### Web API Methods Many Web API functions are defined on the `window` object. Here we use `querySelector()` to find elements on the page. ```js window.document.querySelector('.kitten-pic'); ```
### Finding elements If you are familiar with CSS, `querySelector()` accepts one argument: a string which is a CSS selector. Here we are asking for the first element that has the `class` (that's the period) of `kitten-pic`. ```js window.document.querySelector( '.kitten-pic' ); ``` Which finds this: 
<img class="kitten-pic" src="http://placekitten.com/g/200/200">
### No element found What if the element is not found? `querySelector()` can return `null` if it fails to find an element. ```js window.document.querySelector('.doggo'); // null ```
### Finding multiple elements What if you want to find multiple elements? `querySelectorAll()` returns an array-like object of all the matches. ```js const allSpans = window.document.querySelectorAll( 'span' ); allSpans.length; // 350 ```
### Other Methods There are other functions to find elements. ```js window.document.getElementById( 'my-id' ); window.document.getElementsByTagName( 'span' ); window.document.getElementsByClassName( 'kitten-pic' ); ```
### DOM Nodes The browser keeps an internal representation of what you see on the page. This is called the _Document Object Model_ or DOM.
### DOM Nodes Each element of the page that you access is called a "DOM Node". Here, the variable `cat` holds a DOM Node object. ```js const cat = window.document.querySelector( '.kitten-pic' ); ```
### DOM Node Properties DOM Node objects have properties, like `style`. `style` is an object that holds CSS as properties. ```js const cat = window.document.querySelector( '.kitten-pic' ); cat.style.visibility; // 'hidden' ```
### DOM Node Changes We can change properties on DOM Nodes, and the changes usually take effect immediately. ```js const cat = window.document.querySelector( '.kitten-pic' ); cat.style.visibility; // 'hidden' cat.style.visibility = 'visible'; cat.style.visibility; // 'visible' ```
### CSS Property Names If you want to access a CSS property that has a hyphen in the name, change it to camelCase. ```js // This is for the CSS property `margin-top` cat.style.marginTop; ```
### Changing HTML Properties We can change HTML properties also. ```js const cat = window.document.querySelector( '.kitten-pic' ); cat.src; // 'http://placekitten.com/g/200/200' cat.src = 'http://placekitten.com/g/215/215'; ```
### innerHTML We can actually change all the contents of an element using the `innerHTML` property. ```js const cat = window.document.querySelector( '.cat-saying' ); cat.innerHTML = 'I love you!'; ```
I, the cat, am your master.
### Creating DOM Nodes It's also possible to create new Nodes! `window.document.body` is the Node that contains the HTML body. ```js const cat = window.document.createElement( 'img' ); cat.src = 'http://placekitten.com/g/200/200'; window.document.body.appendChild( cat ); ```
### Creating Text Nodes If you want to create a Node that's just text, use `createTextNode()` ```js const cat = window.document.createTextNode( 'Meow!' ); window.document.body.appendChild( cat ); ```

Events

Photo credit: Eva Peris cc

### Events When something happens on a web page (eg: when a user clicks a button), an _Event_ is created.
### Event Objects Events are objects with information about what happened. There are two ways to listen to events: `element.addEventListener()` in JavaScript and `onClick` (and similar attributes) in the HTML.
### HTML onClick Using HTML we can register a click handler as follows: 
<button onClick="getKitten()">Get Kitten</button>
### onClick handler Here's the JavaScript for that click handler. ```js function getKitten() { const parent = window.document.querySelector( '.get-kitten' ); const kitten = window.document.createElement( 'img' ); kitten.src = 'http://placekitten.com/g/200/200'; parent.appendChild( kitten ); } ```
### Pure JS click handler You can also add the click handler function in pure JavaScript. ```js const button = window.document.querySelector( '.get-kitten-button' ); button.addEventListener( 'click', getKitten ); ``` Notice that we are passing `getKitten` as a variable! We are not calling it here, but sending it to be called later. This is called a "callback".
### Other Events Clicks are common events, but there are many others. - `mouseOver`: when a cursor is moved on to an element - `mouseOut`: when a cursor is moved off of an element - `keyUp`: when a key is released - `load`: when the document finishes loading - `focus`: when an element gets focus - `blur`: when an element loses focus - ...

### Form input You can use `.value` to get the input from a form field. Try this: 
<script>
  function showValue() {
    alert( document.getElementById('myName').value );
  }
</script>
<form>
  <input type="text" id="myName" value="hi" />
  <input type="button" onclick="showValue()" value="click me">
</form>
